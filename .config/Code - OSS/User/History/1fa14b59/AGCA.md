## Packages Used

- boxen
- chalk
- inquirer
- yargs

## Install

- Clone this repo `git clone --depth=1 https://github.com/berthosefin/dotfiles ~/.dotfiles`

```bash
$ git clone https://github.com/berthosefin/cyberto
$ cd cyberto
$ npm install
$ node .
```
